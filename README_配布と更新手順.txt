エアコン販買マスター Pro 配布用正式版 1.1.1

【この版について】
・工事チェック機能は削除済みです。
・ダイキン攻略、接客フロー、練習問題、工事料金、学習記録、図解ライブラリ、工事図解を収録しています。
・iPhone / iPadで利用できます。

【GitHubへの更新方法】
1. このZIPを展開します。
2. GitHubの対象リポジトリを開きます。
3. Add file → Upload files を選びます。
4. 展開した中の全ファイルをまとめてアップロードします。
5. Commit changes を押します。
6. 数分後、公開URLを開き直します。
7. 画面に「正式版 1.1.1」と表示されることを確認します。

【利用者への案内】
更新後は、一度アプリを完全に閉じてから開き直してください。
古い表示が残る場合は、Safariで公開URLを開いて再読み込みしてください。

【注意】
型番・年度・能力帯によって仕様が異なる場合があります。
最終案内は最新カタログ、施工説明書、現地確認を優先してください。


【ホーム画面アイコン】
アイコン修正版です。更新後、古いホーム画面アイコンを一度削除し、Safariから再度「ホーム画面に追加」すると確実に反映されます。
